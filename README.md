# JudeTravel Automation

This project contains automation workflows for AL JUDE Travel & Tourism in Amman, implemented using n8n. It includes 8 specialized agents that handle different aspects of the travel business:

1. Agent 1: Customer Inquiry
2. Agent 2: Flight Booking
3. Agent 3: Hotel & Package
4. Agent 4: Visa Documentation ✅ ACTIVE
5. Agent 5: Marketing & Content
6. Agent 6: Finance & Invoicing ✅ ACTIVE
7. Agent 7: Quality & Feedback ✅ ACTIVE
8. Agent 8: Umrah & Hajj ✅ ACTIVE

Each workflow is stored as a JSON file in the `workflows` directory.# Last updated: Fri Apr 24 20:58:42 UTC 2026
